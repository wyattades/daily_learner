Tutorial

Step 1: Sessions

Each session represents a machine learning instance with its own unique data and model.
You can create a new session by:
  1. Click on Sessions on the right of the tool bar
  2. Click on the + New Sessions button
  3. Give a unique name to your session
  4. Write a description about the content of this session

Step 2: Adding Labels and Selecting a Model

Labels are used to categorize you training data. There are two types of labels, Training Labels and Result Labels.
Training labels will correspond to the different variables you are training on, eg. The different columns of your csv file.
You will only have one result label, this will correspond to the answer, or the number you are trying to predict.
The model type you select will drasticly vary the results you get, Look at the About page to learn about each of them.
  1. Input the label name in the Labels input field
  2. To add the label to your model + button
  3. Repeat previous steps till all of your input labels are listed
  4. Add your Result Label for the variable you are trying to predict
  5. Select your desired model from the Model Type list
Example

Step 3: Adding Training Data

In this step you will add training data that will correspond to the labels you entered when defining the session.
You will have two options to upload training data, Bulk Import or Add Record
Bulk Import will allow you to upload a .csv file.
Add Record will allow you to hand input data.

Bulk Import
  1. On the Sessions page click on the Go to this Session for the session you just created
  2. Press the Bulk Import button
  3. Press the choose a file button
  4. Select a .csv file that as a column for all of your input labels and one for you result label(look below for reference)
  5. Press the import button
  6. Your data should show up on your session's home page
  7. Note that the larger the .csv file, the longer it will take to import

Add Record
  1. On the Sessions page click on the Go to this Session for the session you just created
  2. Press the Add Record button
  3. Fill in the input fields corresponding to each of your input variables
  4. Fill in the input field corresponding to the result variable
  5. Click the Submit button
  6. Repeat these steps until all of your data is imported
  7. Your data should show up on your session's home page

Step 4: Training and Predicting

Once your data is uploaded you will be able to train your model.
When your model is finished training you will be allowed to use it for predictions
It is important to note that if you switch the model type of your session, you will need to retrain it
  1. Click the Train Model button
  2. Note that the training may take some time
  3. The closer the R-Score is to 1, and the smaller the Mean Squared Error, the better the model will preform
  4. If you are seeing poor score results, input more data or change your model type
  5. Once the model is trained to your satisfaction, press the Predict button
  6. Enter input variable values in which you would like your model to predict for
  7. Press the Predict button and your result should be displayed below
